AtlasSystem
===

Atlas system for Unity.

![](http://mob-sakai.github.io/AtlasSystem/images/demo.png)

![](http://mob-sakai.github.io/AtlasSystem/images/inspector.png)


## Requirement

* Unity 5.3+
* No other SDK are required.



## Usage

1. Download the package from [releases](https://github.com/mob-sakai/AtlasSystem/releases).
1. Import the package into your Unity project. Select `Import Package > Custom Package` from the `Assets` menu.

## Use Atlas For UI

1. Add `AtlasImage` component insted of `Image` component.
1. Select altas from exsits.
1. Select an sprite from sprites included in atlas.



## Use Atlas For SpriteRenderer

1. Add `AtlasRenderer` component to the `SpriteRenderer` GameObject.
